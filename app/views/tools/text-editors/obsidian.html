Obsidian

Create a website with Obsidian

Obsidian is a Markdown note-taking app that uses plain files stored on your computer. Because Blot turns folders of files into websites, the two work well togetherâ€”just point Blot at the same folder you use as your Obsidian vault.

Recommended settings

  • Set New link format to relative path to file. Blot also understands absolute path in vault if your vault is the root of your blog folder.
  • Keep your attachments in a folder inside the vault (Obsidian's default attachments location works). Blot mirrors this folder structure when publishing your site.

Blot features for Obsidian users

  • Wikilinks: Blot resolves [[Links]] using the same filename logic as Obsidian. You can customise the link text with the [[Page name|Custom text]] syntax.
  • Embeds: Obsidian's ![[...]] syntax works for images, audio, video, PDFs, and notes. Alt text after the pipe (for example ![[File.png|Caption]]) becomes the published alt attribute.
  • Markdown notes as components: Embedding another note (for example ![[Project overview]]) renders the Markdown inline on your site, so you can reuse content in multiple posts.

See the wikilinks guide for detailed examples of how embeds and link resolution behave when your notes are published with Blot.
