Word Documents

.docx  .odt

Supports files created by Microsoft Word, Pages, Google Docs and OpenOffice. Much formatting is stripped so posts match the style of your site.

Metadata

Add metadata to your document by starting with a block of key: value pairs in the first paragraph or by inserting a YAML front matter block. Blot reads this information the same way it does for plain text posts, so fields like Title, Date and Tags are picked up automatically.

Headings

Blot looks for bold text to determine the structure of the post. A bold line at the top becomes the post title, and additional bold paragraphs are turned into headings so the published page has a sensible outline.

Images and assets

Blot extracts images embedded in the document and adds them directly to the post created from the Word file, so they appear in the same place as they did in your document.

Formatting cleanup

Inline styles and unused tags are removed during conversion. This keeps the post lightweight and lets your site's theme control the final appearance.
